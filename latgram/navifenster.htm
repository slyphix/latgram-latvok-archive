Navigationsfenster

Das Navigationsfenster ist eine Navigationserweiterung.
Es handelt sich hierbei um ein kleines Fenster, das auf Mausklick geöffnet werden kann.
Wenn ein Link im Navigationsfenster angeklickt wird, wird die Seite im Hauptfenster angezeigt.
Zum Schließen des Fensters genügt ein Klick auf "Fenster schließen".
Beachte, dass auch im Navigationsfenster JavaScript aktiviert sein muss, damit die Verweise richtig funktionieren.
